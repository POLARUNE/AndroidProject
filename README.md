### **게임 컨셉**

화면 크기가 900x1600으로 고정인 수박 게임을 만든다.

작은 과일을 합쳐 가장 큰 수박을 만드는 게임이다.



### **개발 범위/핵심 요소**

* 물리 엔진: 과일의 충돌, 탄성, 마찰 구현
* 터치 상호작용: 화면 상단에서 손가락을 움직여 과일 위치를 정하고, 손을 떼면 아래로 떨어뜨림
* 과일 합성: 같은 종류의 과일이 충돌하면 다음 단계의 더 큰 과일로 합성
* 게임 오버 판정: 과일이 지정된 한계선을 넘어서 일정 시간 머무를 경우 종료
* UI/UX: 현재 점수, 최고 점수, 다음에 나올 과일 미리보기

##### 인게임 화면

* 과일 생성: 시스템이 무작위로 작은 크기의 과일 중 하나를 선택해 상단에 배치
* 조준 및 투하: 사용자가 화면을 터치하여 좌우로 드래그하면 과일이 가로로 이동, 손을 떼는 순간, 물리 엔진의 중력이 적용되어 과일 낙하
* 충돌 및 합성: 과일이 바닥이나 다른 과일에 부딪힘, 충돌 지점의 좌표를 계산해 동일한 두 과일을 제거, 그 자리에 한 단계 큰 과일 생성 및 점수 추가
* 연쇄 반응: 합성으로 새로 생긴 과일이 또 다른 동일 과일과 부딪히면 연속으로 합성
* 게임 종료: 모든 물리 연산이 멈춘 상태에서 과일이 상단 한계선 위로 튀어나와 있는지 확인, 일정시간 이상 튀어나와 있을 경우 게임 종료
* 점수 저장: 점수를 저장하고 랭킹에 반영
